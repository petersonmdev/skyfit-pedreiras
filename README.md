# SkyFit Pedreiras

Landing page comercial desenvolvida para o pré-lançamento da SkyFit Unidade Pedreiras, com foco em posicionamento de marca, geração de leads e conversão de interessados para o Grupo VIP e canal de atendimento via WhatsApp.

## Visão Geral

Este projeto foi concebido para sustentar a comunicação de inauguração da unidade SkyFit em Pedreiras-MA, apresentando de forma objetiva os principais diferenciais da academia e conduzindo o público para uma ação direta de interesse comercial.

A página foi estruturada para fortalecer três objetivos centrais:

- comunicar autoridade e percepção de alto padrão da marca;
- destacar a oferta de lançamento para Alunos Fundadores;
- capturar demanda qualificada antes da abertura oficial da unidade.

## Execução

### Requisitos

- Node.js 18+ 
- npm 9+

### Instalação

```bash
npm install
```

### Ambiente de desenvolvimento

```bash
npm run dev
```

### Build de produção

```bash
npm run build
```

### Pré-visualização da build

```bash
npm run preview
```
